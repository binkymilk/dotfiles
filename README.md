## Setting up
Install dotbot.
```shell
./install
```

Load `.bashrc` and `.bash_profile` contents.
```shell
source ~/.bashrc
source ~/.bash_profile
```

For Mac PCs, turn off login in terminal.
```shell
touch .hushlogin
```
